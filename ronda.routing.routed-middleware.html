ronda.routing.routed-middleware

disable-middlewares

(disable-middlewares descriptor route-id ks)

Disable the given middlewares for the endpoint identified by the given route ID.

enable-middlewares

(enable-middlewares descriptor route-id ks)

Enable the given middlewares for the endpoint identified by the given route ID.

routed-middleware

(routed-middleware handler middleware-key wrap-fn & [{:keys [enabled?], :or {enabled? false}}])

Generate a handler that applies the middleware represented by wrap-fn only if it is explicitly enabled/not-disabled for the currently routed endpoint. This has to be used below ronda.routing.middleware/wrap-routing.
